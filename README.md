# 🎮 Pokemon Unity

Proyecto Unity para juego de Pokémon desarrollado por **Isaac Esteban Haro Torres**.

---

## 📝 Descripción

Juego de Pokémon desarrollado en Unity con características de captura, batalla y exploración. Implementa mecánicas clásicas de los juegos de Pokémon.

---

## ✨ Características

- Sistema de combate por turnos
- Captura de Pokémon
- Exploración de mundo abierto
- Sistema de estadísticas y evoluciones
- Gestión de equipo

---

## 🛠️ Stack Tecnológico

- Unity
- C#
- Assets de Pokémon

---

## 📂 Estructura

```
POKEMON-UNITY/
├── Assets/
│   ├── Scripts/        # Código del juego
│   ├── Scenes/         # Escenas del juego
│   ├── Prefabs/        # Objetos predefinidos
│   └── Sprites/        # Gráficos
├── Library/
└── ProjectSettings/
```

---

## 🚀 Uso

1. Abrir con Unity Hub
2. Seleccionar versión 2021.3+ recomendada
3. Importar proyecto
4. Ejecutar escena principal

---

## 🛠️ Requisitos

- Unity 2021.3 o superior
- Visual Studio o VS Code
- Conocimientos de C#
